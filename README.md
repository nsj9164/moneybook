## 📦 주요 기능
- **맞춤형 지출 분류 및 분석**
    - 사용자가 직접 정의한 기준으로 지출을 분류하고 분석하여 자신의 실제 지출 패턴과 범위를 정확히 파악할 수 있습니다.
- **실제 지출 금액 반영**
    - 지출 금액과 실지출 금액을 별도로 입력할 수 있어, 더치페이(1/n 정산)한 금액까지 고려하여 한 달간의 정확한 지출 내역을 확인 가능합니다.
